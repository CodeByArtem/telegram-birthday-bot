# 🖼️ API сервисы изображений

Бот теперь может использовать профессиональные изображения из API сервисов вместо генерации через AI!

## 🚀 Приоритет изображений

1. **API сервисы** (Unsplash, Pexels, Pixabay) - быстрые и качественные
2. **AI генерация** (если API недоступны)
3. **Заглушка** (если всё недоступно)

## 📋 Поддерживаемые API сервисы

### 1. Unsplash API
- **Описание**: Бесплатные высококачественные фотографии
- **Лимит**: 50 запросов в час
- **Качество**: Отличное

### 2. Pexels API  
- **Описание**: Бесплатные стоковые фотографии
- **Лимит**: 200 запросов в час
- **Качество**: Хорошее

### 3. Pixabay API
- **Описание**: Бесплатные изображения и фото
- **Лимит**: 100 запросов в час
- **Качество**: Среднее

## 🔑 Как получить API ключи

### Unsplash API
1. Перейдите на https://unsplash.com/developers
2. Создайте аккаунт/войдите
3. Создайте новое приложение
4. Скопируйте **Access Key**

### Pexels API
1. Перейдите на https://www.pexels.com/api/
2. Создайте аккаунт/войдите
3. Заполните форму для получения API ключа
4. Получите ключ на email

### Pixabay API
1. Перейдите на https://pixabay.com/api/docs/
2. Создайте аккаунт/войдите
3. Скопируйте **API Key** из профиля

## ⚙️ Настройка переменных окружения

Добавьте ключи в `.env` файл:

```env
# Существующие ключи
TELEGRAM_BOT_TOKEN=ваш_токен
TELEGRAM_CHAT_ID=ваш_chat_id
ADMIN_USERNAMES=admin1,admin2
GOOGLE_GEMINI_API_KEY=ваш_gemini_ключ
REPLICATE_API_TOKEN=ваш_replicate_ключ

# Новые ключи для изображений
UNSPLASH_ACCESS_KEY=ваш_unsplash_ключ
PEXELS_API_KEY=ваш_pexels_ключ
PIXABAY_API_KEY=ваш_pixabay_ключ
```

## 🎯 Умные поисковые запросы

Бот автоматически подбирает поисковые запросы в зависимости от:

### Праздника:
- **День рождения**: `birthday`, `celebration`, `cake`
- **8 марта**: `women day`, `spring flowers`, `international women day`
- **Новый год**: `new year`, `celebration`, `winter`

### Стиля:
- **Official**: `elegant`, `formal`, `sophisticated`
- **Funny**: `funny`, `colorful`, `party`, `balloons`
- **Poetic**: `beautiful`, `romantic`, `flowers`, `dreamy`
- **Friendly**: `happy`, `celebration`, `friends`, `warm`
- **Romantic**: `romantic`, `candles`, `intimate`, `roses`

### Примеры запросов:
- `elegant birthday celebration formal cake`
- `funny birthday party colorful balloons`
- `beautiful spring flowers women day`
- `romantic new year candles`

## ✅ Преимущества API изображений

### 🚀 Скорость:
- Мгновенная загрузка (1-3 секунды)
- Нет очередей на генерацию

### 🎨 Качество:
- Профессиональные фотографии
- Высокое разрешение
- Разнообразие стилей

### 💰 Стоимость:
- Бесплатные тарифы достаточны
- Нет расходов на GPU

### 🔧 Надежность:
- Стабильные API
- Множественные бэкапы
- Автоматический фолбэк

## 🚨 Ограничения

### Unsplash:
- 50 запросов/час
- Нужно указывать автора в некоторых случаях

### Pexels:
- 200 запросов/час  
- Нужно указывать источник

### Pixabay:
- 100 запросов/час
- Бесплатно для коммерческого использования

## 🔄 Резервирование

Если все API сервисы недоступны, бот автоматически переключается на:
1. AI генерацию изображений
2. Цветные заглушки

Это гарантирует 100% работоспособность! 🎉
