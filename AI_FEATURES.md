# 🤖 AI Функции бота

## 📋 Обзор

Бот теперь поддерживает генерацию праздничных поздравлений с помощью искусственного интеллекта:

- **🧠 Google Gemini** - генерация текстовых промптов и поздравлений
- **🎨 Stable Diffusion** - генерация уникальных изображений
- **🔄 Резервные механизмы** - работа даже если AI недоступен

## 🚀 Настройка

### 1. Получение API ключей

#### Google Gemini (бесплатно)
1. Перейдите на [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Создайте новый API ключ
3. Скопируйте ключ

#### Hugging Face (бесплатно)
1. Зарегистрируйтесь на [Hugging Face](https://huggingface.co/)
2. Перейдите в Settings -> Access Tokens
3. Создайте новый токен
4. Скопируйте токен

### 2. Настройка переменных окружения

Добавьте в `.env.production`:

```bash
GOOGLE_GEMINI_API_KEY=ваш_gemini_api_ключ
HUGGINGFACE_API_KEY=ваш_huggingface_api_ключ
```

## 🎯 Новые команды

### `/ai_status`
Проверяет статус AI сервисов
```
/ai_status
```

### `/ai_test`
Тестирует AI генерацию поздравления с 8 марта
```
/ai_test
```

### `/ai_birthday @username`
Генерирует персональное поздравление с днем рождения
```
/ai_birthday @username
```

### `/ai_holiday [название]`
Генерирует поздравление для любого праздника
```
/ai_holiday Новый год
/ai_holiday Рождество
/ai_holiday Пасха
```

## 🎨 Как это работает

### 1. Генерация промпта
Google Gemini создает детальный промпт для Stable Diffusion на основе:
- Названия праздника
- Имя получателя (если указано)
- Базовых шаблонов для каждого праздника

### 2. Генерация изображения
Stable Diffusion создает уникальное изображение (512x512px) на основе промпта

### 3. Генерация текста
Google Gemini создает красивое поздравление с эмодзи

### 4. Отправка в Telegram
Изображение и текст отправляются как одно сообщение

## 🔄 Резервные механизмы

### Если AI недоступен:
- **Промпты**: используются заранее подготовленные шаблоны
- **Изображения**: создаются простые заглушки с Canvas
- **Текст**: используются готовые поздравления

### Изображения-заглушки включают:
- Праздничный фон
- Текст поздравления
- Декоративные элементы (цветы, снежинки, шары)

- Изображения сохраняются локально и очищаются каждые 24 часа
- Все запросы к AI логируются для мониторинга
